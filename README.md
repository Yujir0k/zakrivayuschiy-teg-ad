# Проектная работа «Закрывающий тег»

Адаптивная страница о пройденных этапах обучения в Яндекс Практикуме.

## Возможности

- восемь карточек с изображениями и CSS-фильтрами;
- анимированные кнопки Like;
- модальное окно сохранения через элемент `dialog`;
- адаптивная вёрстка для мобильных экранов;
- вариативный шрифт Inter и акцентный шрифт PressStart2P.

## Технологии

HTML, CSS, SVG и JavaScript.
